## 项目目录结构分析
```text
--src
    --assets
        --css # 封装的css
        --img # 静态图片
        --js  # js存放位置
    --components # 组件存放位置
    --router  # 路由
        --index.js
    --utils # 工具类
    --views # 各个页面的首页


```
